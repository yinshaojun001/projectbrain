# ProjectBrain Project Context

ProjectBrain is a local-first project cognition layer for AI coding agents.

The project exists to help AI agents safely work on private or complex repositories by giving them scoped project context, impact analysis, human experience claims, and privacy-preserving local MCP tools.

Current public repository:

```text
https://github.com/yinshaojun001/projectbrain
```

Core differentiation:

- local-only stdio MCP server
- no hosted ProjectBrain dependency
- no source code upload
- Context Pack before editing
- Impact Analysis before commit
- human project experience as first-class knowledge
- future stale knowledge lifecycle

Primary competing reference:

- Understand-Anything: strong codebase graph explorer/dashboard/onboarding product.

ProjectBrain should avoid copying a dashboard-first product and instead focus on AI agent execution safety.
